# ovos-docs-viewer

Read the Open Voice OS and HiveMind documentations from your terminal!

```
Usage: ovos-docs-viewer [OPTIONS] DOCS

  launch docs viewer, choose one of 'community', 'technical', 'hivemind',
  'messages'

Options:
  --help  Show this message and exit.
```
